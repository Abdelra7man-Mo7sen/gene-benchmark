# Paper Summary 

**Title:** Large-scale empirical benchmark of 11 automated gene identification tools

**Main Question:** Which bioinformatics tool is best for finding genes in DNA sequences?

**Methods:** Tested on 113 genomes (bacteria, human, plants, fungi) using 4 validation techniques (RNA-seq, proteomics, synteny, known annotations).

**Key Findings:**
- Prodigal: Best for complete bacterial genomes (98.7% accuracy)
- MetaGeneAnnotator: Best for fragmented metagenomes (91.4%)
- BRAKER2: Best for eukaryotes with RNA-seq (90.1%)
- Ensembles (combining tools): 99.6% accuracy for bacteria
